right float

Right-floated box sits at the right edge of the body content area; this paragraph wraps along the LEFT edge until it clears the float bottom. CSS 2.1 §9.5.1 specifies the floated box anchors to the closest preceding line and following text flows around it.

left float

Left-floated box anchors to the LEFT; this text flows along the right side. Multiple paragraphs in a row keep wrapping until they pass the float vertical extent, then they widen back to full content width.

A3 unit resolution

[baseline] no margin, light-blue background. Wraps to fill content area.

[margin-left:1em] over 16px parent = 16px indent.

[margin-left:2em] over 16px parent = 32px indent.

[margin-left:1rem] = 16px indent (root html font-size).

[padding:1em] all sides = 16px padding.

[padding-left:50px] = 50px indent inside the box.

[width:50%] half viewport width.

[width:200px] fixed pixel width.

[em on 24px ancestor] margin-left:1em should be 24px since element's font-size inherits 24.

[0.5em over 24px] = 12px indent.

[em on 12px ancestor] margin-left:1em should be 12px (smaller indent).

[margin:0.5em 4em + padding:0.5em 1em] mixed em on shorthand. 0.5em=8, 4em=64, 1em=16 vs 16px parent.

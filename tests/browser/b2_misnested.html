B2 misnested formatting tags

[case 1] Plain. Bold. Both. ItalicOnly. Plain.

[case 2] Plain. Italic. Both. BoldOnly. Plain.

Spec: in case 2, "BoldOnly." should still be bold even though </i> came first (adoption-agency keeps <b> open across </i>). With the current parser, both <i> and <b> get popped at </i>, so "BoldOnly." may render as plain instead.

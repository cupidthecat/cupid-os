# Bound CupidDis to the Cupid OS machine-code domain

CupidDis will provide objdump-class inspection with DWARF v4 source information and decode 16-bit and 32-bit x86 through x87, MMX, and SSE2; x86-64 and later instruction families are outside its scope. This fully covers the intended Cupid OS execution domain while preventing an open-ended ISA and debug-format compatibility project.

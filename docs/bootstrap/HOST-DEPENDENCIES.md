# Host dependency inventory

The deterministic active-source audit records three supported build roots: root `all`, `user:all`, and `toolchain:all`. `audits/active-build.json` owns the current 694-source/498-transform graph and its 432-path manifest covers all 425 final-link objects. The language graph contains 271 C translation units, 270 headers, and 127 Cupid C files; the prioritized toolchain-source cohort contains 65 files. That evidence predates the CupidC preprocessing, declaration, type/layout, IR, and object contracts. The current graph declares eighteen hosted artifacts, so its complete three-root cohort would contain 453 logical artifacts when recaptured from one committed revision.
